# 🔩 Knuckle Joint Design & Assembly

## 📌 Project Description
This project involves the CAD design and assembly of a knuckle joint using SolidWorks. It demonstrates key mechanical components, proper mating, and motion. The model highlights tensile load applications and provides a foundation for further analysis like stress and fatigue testing.

## ⚙️ Components
- Eye End  
- Fork End  
- Knuckle Pin  
- Collar  
- Taper Pin  

## 🛠️ Tools Used
- SolidWorks  

## 📂 Files
- Assem2.SLDASM (Assembly File)  
- Assem2.mp4 (Animation)  

## 🚀 Usage
Open the assembly file in SolidWorks and run the animation to visualize the working of the knuckle joint.

## 📈 Future Scope
- Stress Analysis (FEA)  
- Fatigue Analysis  
- Design Optimization
